# Afinidad con Luna Frontend

Vue 3 + Vite virtual pet app focused on Luna's affinity, memories, and gentle petting.

## Environment

Copy `.env.example` to `.env` when the API is not served from the same origin.

```env
VITE_API_URL=http://localhost:3000
```

Optional audio files can be added later:

- `public/sounds/ronroneo.mp3`
- `public/sounds/despertar.mp3`
